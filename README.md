# Page Detective

WebページのQAチェックを自動化するChrome拡張機能です。  
リンク切れ・画像ALT抜け・Meta タグ・見出し構造・読み込み速度・SEO/GEO など、複数の項目を一括でチェックし、結果をサイドパネルまたは専用レポートページで確認できます。

---

## 動作環境

- Google Chrome 116 以降

---

## インストール方法

### Chrome ウェブストアからインストール（推奨）

Chrome ウェブストアのページから「Chromeに追加」をクリックしてください。

### 手動インストール（開発者向け）

1. `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、本フォルダを選択する

---

## 使い方

1. チェックしたい Web ページを Chrome で開く
2. ツールバーの **Page Detective アイコン** をクリックしてサイドパネルを開く
3. チェックしたい項目にチェックを入れ、**「チェック実行」** ボタンを押す
4. 結果がサイドパネル内に表示され、問題のある要素がページ上でハイライトされる
5. **「タブで開く」** ボタンを押すと、詳細なレポートページが別タブで開く
6. 結果は **CSV** または **Excel** 形式でダウンロードできる

### チェック項目

| 項目 | 内容 |
| ------ | ------ |
| 画像 alt | `alt` 属性が欠落または空の画像を検出 |
| Meta タグ | title・description・viewport・OGP の有無と文字数を検証 |
| 無効リンク | `href` が空・`#` のみのリンクを検出 |
| 見出し構造 | h1 の数・空見出し・見出しレベルのスキップを検出 |
| HTTP チェック | ページ内リンクへ実際にリクエストし 200 以外を検出 |
| 読み込み速度 | Navigation Timing API で読み込み時間を計測 |
| SEO / GEO | canonical・robots・JSON-LD 構造化データなどを確認 |

### ページ上のハイライト表示

| 色 | 対象 |
| ---- | ------ |
| 赤枠 | alt 属性が欠落・空の画像 |
| オレンジ枠 | href が無効なリンク |
| 紫枠 | 問題のある見出し |
| 青枠 | HTTP エラーのリンク |

---

## 注意事項

- `chrome://` ページなど一部のシステムページではチェックを実行できません
- HTTP チェックはサーバー側の CORS 設定によって正確なステータスを取得できない場合があります
- 読み込み速度の計測は、ページが完全に読み込まれた後に実行してください
